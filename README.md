Training Repository

This repository contains all the training materials, tasks, source code, and practice exercises covered during the sessions.

Folder Structure
HTML/ → HTML practice files and assignments
CSS/ → CSS concepts, layouts, Flexbox, Grid, and tasks
JavaScript/ → JavaScript basics, DOM tasks, and mini projects
Projects/ → Combined frontend practice projects
Assets/ → Images, icons, and supporting files
